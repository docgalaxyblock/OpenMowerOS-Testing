!!!!! HEADS-UP WINDOWS USERS !!!!!
----------------------------------

Do not use Wordpad or Notepad for editing files here,
it will mangle them and your configuration won't work.
Use a proper text editor instead (Notepad++, VSCode, Atom or SublimeText).

!!!!! HEADS-UP MacOS USERS !!!!!
---------------------------------

If you use Textedit to edit the files make sure to use "plain text format"
and "disable smart quotes" in "Textedit > Preferences", otherwise Textedit
will use none-compatible characters and your network configuration won't work!
